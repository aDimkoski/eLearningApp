Welcome to the Course: Deep Learning

 

Overview

The purpose of this learning unit is to introduce the basic functionalities of Deep Learning. The Deep Learning Specialization is a foundational program that will help you understand the capabilities, challenges, and consequences of deep learning and prepare you to participate in the development of leading-edge AI technology.

In more detail, the learning unit discusses the following topics:

  • What is Deep Learning
  • Difference between Neural nets vs Deep Learning
  • How Neural nets learn
  • Back propagation
  • Gradient descent
  • Activation functions
  • Different types of NN in Deep Learning
