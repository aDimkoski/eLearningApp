Welcome to the Course: Machine Learning

 

Overview

The purpose of this learning unit is to introduce the basic functionalities of Machine Learning. Understanding ML practically will guarantee you a secure and lucrative career in the tech industry. With an extensive ML skill set, you can provide great value to your employer and boost your relevance in the job marketplace.

In more detail, the learning unit discusses the following topics:

  • What is Machine Learning
  • Machine Learning Algorithms and Techniques
  • Relation between Artificial Intelligence and Machine Learning
  • Artificial Neural Networks and their applications
  • Reinforcement Learning and Deep Learning
  • Natural Language Processing
  • Time Series Analysis
